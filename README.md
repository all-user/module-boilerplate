# browser-boilerplate

Boilerplate for `browserify`, `babel`, `jade`, `stylus`, `testem`, `mocha`, `power-assert`, `eslint`, `browser-sync` project.

## Install
```shell
$ git clone git@github.com:all-user/browser-boilerplate.git ./my-project
$ cd my-project
$ npm i
```

## Display Tasks
```shell
$ npm run help
```
